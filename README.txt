Como ejecutar el proyecto:

Si quieres correr los casos de pruebas a traves de Cypress UI, realiza los siguientes pasos: 

1. Clona y Abre el projecto usando VS Code 
2. Abrir Git Bash Terminal
3. Ejecutar el comando: npm install
4. Ejecutar el comando: npm run cypress:open
5. Cypress UI se desplegara
6. Seleccionar el archivo 'spec.cy.js'
7. Casos de pruebas se ejecutaran

Para correr los casos de pruebas solo a traves de la consola, realiza los siguientes pasos:

1. Clona y Abre el projecto usando VS Code 
2. Abrir Git Bash Terminal
3. Ejecutar el comando: npm install
4. Ejecutar el comando: npm run cypress:runChrome
